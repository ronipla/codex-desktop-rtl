# Security Policy

## Reporting Security Issues

Please report security issues privately before publishing exploit details.

Include:

- Windows version.
- Codex Desktop version.
- Codex Desktop RTL version/hash.
- Reproduction steps.
- Expected and actual behavior.

## Supported Versions

Only the latest release is supported.

## Security Notes

See `docs/SECURITY.md` and `docs/THREAT_MODEL.md` for the current security posture and known risks.
